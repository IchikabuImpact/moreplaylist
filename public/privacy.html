Privacy Policy

MorePlaylist が Google ユーザーデータをどのように扱うかを説明します。

More Playlist は、アプリケーションをご利用の際に YouTube API を通じてプレイリストへのアクセス許可を求めることがあります。この許可により、ユーザーがアップロードしたプレイリスト情報を取得します。

ただし、ユーザーから提供された情報を第三者に提供したり、意図された用途以外に使用したりすることはありません。

情報の安全性とプライバシーを確保するため、SSL 暗号化による通信を行い、適切な技術的・組織的対策を講じています。

More Playlist は、アプリケーションにリンクされている第三者のサイトやサービスのプライバシー慣行について責任を負いません。

ご質問がある場合は、お問い合わせフォームからご連絡ください。

Google ユーザーデータの取り扱い

MorePlaylist は YouTube API を使用してアカウントを認証し、ログイン認証に利用したデータをデータベースなどのストレージに保持しません。また、二次利用もしません。

Restricted Scope および Sensitive Scope で取得した Google ユーザーデータについては、Limited Use Policy に準拠します。

Google API Services User Data Policy

ユーザーが承認した時点で明示されていない種類のユーザーデータにアクセスする必要がある場合は、プライバシーポリシーを更新し、ユーザーの同意を得ます。

アプリが使用する API スコープ

More Playlist アプリを利用する際、YouTube API を通じてプレイリストへのアクセス許可を求める場合があります。取得した情報はユーザーの目的以外には使用しません。

https://www.googleapis.com/auth/youtube

https://www.googleapis.com/auth/youtube.force-ssl

API スコープの利用目的

  • アカウント内のプレイリストの読み取り。
  • プレイリスト名と ID を取得し、アプリのプルダウンに表示。
  • これらの処理は google-api-php-client を利用して実装。
  • 取得した認証情報はセッション内のみに保存し、セッション終了時に破棄。

google-api-php-client

ユーザーデータの保護

ユーザーデータはデータベースに保存せず、SSL 通信で保護します。
