Privacy Policy

MorePlaylist が Google ユーザーデータおよび利用者情報をどのように取り扱うかを説明します。 広告表示・将来の広告非表示（有料）機能に伴い、必要最小限のユーザー情報をデータベースに保存します。

最終更新日：2026-02-10

MorePlaylist は、アプリケーションをご利用の際に YouTube API を通じてプレイリストへのアクセス許可を求めることがあります。 この許可により、ユーザーのプレイリスト情報（例：プレイリスト名やID等）を取得し、アプリの機能提供に利用します。

当サービスは、ユーザー情報を第三者に販売しません。広告配信やインフラ運用などのために、必要な範囲で外部サービスを利用（委託）する場合があります。

情報の安全性とプライバシーを確保するため、SSL/TLS による暗号化通信を行い、適切な技術的・組織的対策を講じます。

MorePlaylist は、アプリケーションにリンクされている第三者のサイトやサービスのプライバシー慣行について責任を負いません。

ご質問がある場合は、 お問い合わせフォーム からご連絡ください。

収集する情報

  • Google ログインにより取得する基本情報
    例：Google アカウントの一意ID（sub相当の識別子）、メールアドレス、表示名、プロフィール画像URL（提供される場合）
  • アカウント管理のために保存する情報（必要最小限）
    例：アカウント作成日時、最終ログイン日時、利用プラン（広告あり/広告なし）、同意/設定フラグ
  • 利用状況・技術情報
    例：アクセスログ（IPアドレス、User-Agent、リクエスト日時）、Cookie/類似技術により生成される識別子（広告・不正防止・安定運用のため）
  • YouTube API を通じて取得する情報
    例：ユーザーのプレイリスト名やID等（アプリ内表示・操作のために利用）

※当サービスは、Google API へのアクセスに用いる認可情報（アクセストークン/リフレッシュトークン等）をデータベースに保存しません。 これらはセッション内でのみ取り扱い、セッション終了時に破棄します。

広告・Cookie / 類似技術について

当サービスは、広告表示により収益を得る場合があります。広告配信のため、第三者の広告配信事業者が Cookie、Web Beacon、または類似技術を使用し、 ユーザーのアクセス情報等を収集する場合があります。

パーソナライズ広告（興味関心に基づく広告）を表示する場合、各事業者が提供するオプトアウト手段を利用できることがあります。 例：Google の広告設定、または複数事業者のオプトアウト（DAA WebChoices 等）。

なお、ブラウザ設定により Cookie を無効化することも可能ですが、機能の一部が利用できなくなる場合があります。

第三者提供・委託

当サービスは、ユーザー情報を第三者に販売しません。 ただし、以下の場合に限り、必要な範囲で情報を共有/委託することがあります。

  • 広告配信（広告事業者）
  • ホスティング、監視、ログ保管等のインフラ運用（委託先）
  • 不正防止やセキュリティ対策（委託先）

法令に基づく開示要請がある場合、または権利侵害・不正利用への対応のために必要な場合、適法に対応します。

保存期間

当サービスは、利用目的の達成に必要な期間、ユーザー情報を保存します。 退会後も、不正防止・紛争対応・会計監査等のために一定期間保持する場合があります。 アクセスログ等は、運用上必要な期間に限り保持します。

Google ユーザーデータの取り扱い

MorePlaylist は YouTube API を使用してアカウントを認証し、ユーザーのプレイリスト情報を取得します。 取得したプレイリスト情報は、ユーザーが意図したアプリ機能（表示・操作等）の提供のためにのみ利用します。

当サービスは、認可情報（アクセストークン/リフレッシュトークン等）をデータベースに保存しません。 認可情報はセッション内のみに保存し、セッション終了時に破棄します。

一方で、広告表示および将来の広告非表示（有料）機能の提供、アカウント識別のために、 ユーザー識別・プラン管理に必要な最小限の情報をデータベースに保存します。

Restricted Scope および Sensitive Scope に該当する Google ユーザーデータの取り扱いについては、Limited Use Policy に準拠します。

アプリが使用する API スコープ

More Playlist アプリを利用する際、YouTube API を通じてプレイリストへのアクセス許可を求める場合があります。取得した情報はユーザーの目的以外には使用しません。

https://www.googleapis.com/auth/youtube

https://www.googleapis.com/auth/youtube.force-ssl

API スコープの利用目的

  • アカウント内のプレイリストの読み取り。
  • プレイリスト名と ID を取得し、アプリのプルダウンに表示。
  • これらの処理は google-api-php-client を利用して実装。
  • 取得した認可情報（アクセストークン等）はセッション内のみに保存し、セッション終了時に破棄。

google-api-php-client

ユーザーデータの保護

通信は SSL/TLS により暗号化されます。アクセス制御、ログ監視等により、ユーザーデータの安全性確保に努めます。

ただし、インターネット通信に完全な安全は存在しないため、当サービスは合理的な範囲での対策を行います。
